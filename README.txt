Allow customers to register their WooCommerce products using serial numbers.

== Description ==

WooCommerce Product Registration is a plugin that allows customers to register their purchased products using serial numbers. This plugin integrates seamlessly with WooCommerce to provide a smooth product registration experience.

Features:
* Customers can register products using serial numbers
* Admin panel for managing serial numbers and registrations
* Import/export functionality for serial numbers
* Email notifications for successful registrations

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-product-registration` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->WooCommerce Product Registration screen to configure the plugin.

== Frequently Asked Questions ==

= How do customers register their products? =

Customers can use the [product_registration_form] shortcode on any page to access the registration form.

= Can I import multiple serial numbers at once? =

Yes, you can import serial numbers in bulk using a CSV or XLSX file from the admin panel.
